Technical Guide

A deeper look at how the LG Collared Report Workflow is built: the EarthRanger-sourced study area layers, home range and track methodology, summary metrics, and report assembly.

Download the full Technical Guide (PDF) v1.1

Version 1.1 corrects extensive drift from an earlier spec revision. The most significant fix: the guide previously described a "Conflict Hotspot Areas" layer downloaded from Dropbox, alongside a Group Ranch Boundaries file — neither reflects the current spec. There is no conflict-hotspot layer at all; both study-area layers (Conservancies and Group Ranch Boundaries) are fetched live from EarthRanger spatial features, not downloaded from Dropbox. Several task/function names were also wrong throughout (cover page, per-grouper rendering, document merge, view-state, and screenshot tasks), the cover page filename was wrong (cover_page.docx, not lg_cover_page.docx), a "totals row" on the summary CSV was fabricated, and the package list is updated for the ecoscope-platform migration.
